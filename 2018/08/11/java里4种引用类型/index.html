引用类型分类

1.强引用
2.软引用
3.弱引用
4.虚引用

引用类型演示

1.强引用

通过new关键字创造的引用为强引用,当GC进行垃圾收集的时候,只有发现没有任何引用指向该对象才可以进行垃圾回收.
2.软引用

当GC进行垃圾回收时,如果这个时候,JVM内存充足,就不对该对象进行回收,否则进行回收.
3.弱引用

每次进行GC都会回收弱引用.
4.虚引用

如果一个对象只具有虚引用，那么它就和没有任何引用一样，随时会被jvm当作垃圾进行回收，虚引用主要是用于跟踪一个对象何时被GC回收.
