AtomicInteger源码分析

发表于 2018-09-02 | 分类于 源码分析
字数统计: | 阅读时长 ≈

概述

AtomicInteger是jdk1.5之后新增的一个可以原子更新的Integer类.常用来做多线程环境下的计数器.

成员变量

可以看到共包含4个成员变量:
1.value变量存放实际存储的值
2.unsafe变量为sun.misc.Unsafe类的对象,该对象是AtomicInteger实现原子操作的关键.
可以参考以下链接学习一下:
https://blog.csdn.net/u011392897/article/details/60365145
3.valueOffset为通过unsafe计算出的value在内存中的偏移量.
4.serivalVersionUID是序列化相关的版本信息.

构造方法

一个无参数构造器,会初始化value=0,有参数构造器会初始化value为参数值.

api分析

get方法

普遍的获取值得方法.

set方法

普遍的设置值得方法.

lazySet方法

性能比set方法好,但不保证立即对其他线程可见.

getAndSet方法

通过调用unsafe对象里的方法实现原子的设置值,并返回旧的值.

compareAndSet方法

仅当当前值等于期望值的时候,才重新设置值.

weakCompareAndSet方法

从代码角度看和compareAndSet是一样的.

getAndIncrement方法

原子的自增1.

getAndDecrement方法

原子的自减1.

getAndAdd方法

原子的增加一个指定的值.

incrementAndGet方法

原子的增加1,并返回更新后的值.

decrementAndGet方法

原子的自减1,并返回更新后的值.

addAndGet方法

原子的增加一个指定的值,并返回更新后的值.

getAndUpdate方法

使用参数传入的函数接口来更新当前值,更新失败会重试,返回更新之前的值.

updateAndGet方法

根据传入的函数接口更新当前值,并返回更新后的值.

getAndAccumulate方法

根据传入的int值和函数接口,计算更新后的值,并返回更新前的值.

accumulateAndGet方法

通过传入的int值和函数接口,更新当前值,并返回更新后的值.

toString方法

返回当前值得字符串表示.

intValue方法

返回当前值的int表示.

longValue方法

返回当前值的long表示.

floatValue方法

返回当前值的float表示.

doubleValue方法

返回当前值的double表示.
